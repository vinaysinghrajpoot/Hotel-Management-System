# 🏨 Hotel Management System (Java Desktop Application)

A professional, feature-rich management suite developed to automate hotel operations, enhance guest services, and manage staff records. This application is built using **Java (J2SE)** and utilizes **MySQL** for robust data persistence.

---

## 🌟 Key Modules & Features

### 🔐 Security & Access
* **Encrypted Login:** Secure login system for administrators to prevent unauthorized access.
* **User Registration:** Professional signup panel for new staff members.

### 🏩 Resource Management
* **Room Inventory:** Dedicated module to add, view, and manage room types (Single, Double, Suite) and pricing.
* **Food & Menu Management:** Dynamic menu management system to handle restaurant/room service orders.

### 👤 Operations & Staffing
* **Customer Records:** Maintain detailed guest profiles, check-in/out logs, and contact information.
* **Employee Hub:** Manage employee profiles, roles, and salary details.
* **Attendance Tracking:** Built-in attendance module for daily staff monitoring.

### 💳 Billing & Reports
* Automated calculation of room charges and food bills for seamless checkout.

---

## 🛠️ Technical Stack

* **Front-End:** Java Swing, AWT (Custom modern UI components)
* **Back-End:** Java (J2SE)
* **Database:** MySQL (Relational Database Management)
* **Connectivity:** JDBC (Java Database Connectivity)

---

## 📂 Repository Structure
* `Admin_Home.java`: The central dashboard for all operations.
* `Add_Room.java` / `Add_Food.java`: Modules for inventory management.
* `attendance.java`: Specialized class for staff management.
* `customer.java`: Logic for guest data handling.

## 🚀 Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/vinaysinghrajpoot/Hotel-Management-System.git](https://github.com/vinaysinghrajpoot/Hotel-Management-System.git)
    ```
2.  **Database Configuration:**
    * Import the provided SQL schema into your MySQL server.
    * Update the database URL, username, and password in the connection logic.
3.  **Library Requirements:**
    * Add `mysql-connector-java.jar` to your project's build path.
4.  **Run Application:**
    * Execute `loginpain.java` to launch the authentication screen.
